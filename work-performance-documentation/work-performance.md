# Work performance

- Examples:
  - Work completed
  - Key performance indicators (KPIs)
  - Technical performance measures
  - Actual start and finish dates of schedule activities
  - Story points completed
  - Deliverables status
  - Schedule progress

  - Number of defects
  - Actual costs incurred
  - Actual durations
